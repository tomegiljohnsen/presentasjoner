# Presentasjoner

Animerte HTML-presentasjoner om Amway og Yager-systemet — historikk, prinsipper og menneskene som har formet en hel bransje.

## Kom i gang

Åpne `index.html` i nettleseren — eller besøk live-versjonen via GitHub Pages.

## Innhold

- **Amway** (`amway.html`) — Stifterne Rich DeVos & Jay Van Andel, FTC-dommen i 1979, økologiske farmer og dagens ledelse. 14 scener.
- **Yager-systemet** (`yager.html`) — Dexter & Birdie Yager, nivåene Amway måtte bygge for ham, de fire pilarene og videreføringen. 13 scener.

## Hvordan det fungerer

Landingssiden gir to valg. Når du tapper et kort, sender en smart router deg til riktig versjon basert på enheten din:

- **PC/desktop** → fullskjermsversjon med horisontal tidslinje
- **Mobil/nettbrett** → vertikal layout optimert for touch og sveip

## Navigering

- **Klikk eller tap** for å gå til neste scene
- **Sveip** venstre/høyre på mobil
- **Pause/play**-knapp i kontrollinjen nederst
- **Piltaster** ← → eller `mellomrom` på PC
- **P** pauser

## Filstruktur

```
index.html              ← Landingsside (velg presentasjon)
amway.html              ← Smart router (sender til riktig Amway-versjon)
yager.html              ← Smart router (sender til riktig Yager-versjon)
amway-historien.html    ← Amway PC-versjon
amway-mobil.html        ← Amway mobilversjon
yager-systemet.html     ← Yager PC-versjon
yager-mobil.html        ← Yager mobilversjon
```

## Et duplikerbart verktøy

Disse presentasjonene er laget for å brukes fritt av hvem som helst i nettverket. Innholdet inneholder ingen personlige detaljer — bare lenker, fakta og budskapet.
